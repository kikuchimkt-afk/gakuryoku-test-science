# 学力テスト問題検索 テンプレート

このフォルダには、新しい教科用のデータベースアプリを作成するためのテンプレートファイルが含まれています。

## 使い方

### 1. 新しいフォルダを作成
```
例: c:\Users\makoto\Documents\アプリ開発\学力テスト社会検索\
```

### 2. このtemplateフォルダの中身をすべてコピー
- `学力テスト検索.html` → 検索画面
- `問題登録.html` → 登録画面
- `start_server.bat` → ローカルサーバー起動用
- `setup_subject.bat` → 教科名設定用 ⭐NEW
- `database/index.json` → データベース（空）

### 3. 教科名を設定（簡単！）

**`setup_subject.bat` をダブルクリック** するだけ！

教科名を入力すると、すべてのHTMLファイル内の【教科名】が自動で置換されます。

```
例: 社会、数学、英語 など
```

### 4. ファイル名の変更（任意）
- `学力テスト検索.html` → `学力テスト社会検索.html`

### 5. 使用開始
1. `start_server.bat` をダブルクリック
2. ブラウザで `http://127.0.0.1:8080/問題登録.html` を開く
3. 問題を登録

## 作成する教科の例
- 学力テスト社会検索
- 学力テスト数学検索
- 学力テスト英語検索
